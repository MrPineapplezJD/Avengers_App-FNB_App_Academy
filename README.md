# 🦸 Avengers Web App

## 📖 Overview
This project was developed during the **FNB App Academy** program.  
It is a simple web application showcasing the **Avengers characters** with their bios and power stats.  
The app demonstrates fundamental **HTML, CSS, and file structuring** skills, while practicing clean navigation and styling.

---

## ✨ Features
- 🧑‍🎤 **Character Pages**: Each Avenger has a dedicated HTML page with bio and stats.
- 🖼️ **Gallery Navigation**: The homepage (`index.html`) displays images linking to each hero’s profile.
- 🎨 **Consistent Styling**: A shared stylesheet (`styles.css`) ensures a unified look across all pages.
- 📂 **Data Integration**: A text file (`avengersInfo.txt`) contains structured information about each hero.

---

## 📂 File Structure
```
project-root/
│
├── index.html              # Homepage with gallery of Avengers
├── about.html              # About page describing the project/company
├── contact.html            # Contact page with developer details
├── thor.html               # Thor’s profile
├── ironman.html            # Iron Man’s profile
├── hulk.html               # Hulk’s profile
├── hawkeye.html            # Hawkeye’s profile
├── captain-america.html    # Captain America’s profile
├── black-widow.html        # Black Widow’s profile
├── avengersInfo.txt        # Text-based data of all Avengers
├── styles.css              # Shared stylesheet
└── images/                 # Folder containing Avengers images
```

---

## 🚀 How to Run
1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. Navigate through the menu or gallery to explore each Avenger’s profile.
